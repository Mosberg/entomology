Key Features

Gameplay

  • 8 custom insects: Beetle, Butterfly, Cicada, Damselfly, Firefly, Fly, Monarch Butterfly, Mosquito.
  • 5 bug net tiers with 60%–100% catch rates.
  • Specimen jars, research station, display cases, and field guide.
  • Breeding mechanics, environmental preferences, and advancements.

Technical

  • Modular, data‑driven architecture with JSON configuration.
  • Schema validation and hot‑reloadable configs.
  • Custom render states, HUD overlay, particles, and keybinds.
  • Public API, component system, and test suite with 24 tests.

Gameplay Systems

Insects & Capture

Use bug nets in five tiers to capture insects into specimen jars, each storing rarity, size, and condition data.

Research & Field Guide

Study specimens at the research station to unlock detailed entries in the in‑game field guide.

Breeding & Environment

Breed compatible species using advanced, schema‑driven rules that respect biome, time, and climate preferences.

Collection & Display

Showcase your collection in display cases and track progress with a full advancement set.

Technical Overview

Architecture

The mod separates core, entity, client, item, block, mechanics, config, data, API, schema, and testing layers for clarity and extensibility.

Data‑Driven Design

Specimens, research entries, mechanics, and more are defined via JSON and validated against schemas for robust modpack integration.

Installation

Players

  1. Install Fabric Loader 0.18.1+ for Minecraft 1.21.10.
  2. Install Fabric API 0.138.3+ for the same version.
  3. Drop the Entomology jar into your .minecraft/mods/ folder.
  4. Launch Minecraft and enjoy collecting insects.

Developers

  1. Clone the repository.
  2. Run ./gradlew build to compile.
  3. Use ./gradlew runClient and runServer for dev.
